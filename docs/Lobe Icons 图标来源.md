# Lobe Icons 图标来源

本项目的 AI 模型与供应商图标来自开源项目 `lobehub/lobe-icons`。

- 项目地址：https://github.com/lobehub/lobe-icons
- 许可协议：MIT License
- 使用方式：下载静态 PNG 图标并保存为 Android 本地资源，避免运行时依赖网络。

当前接入图标：

- OpenAI
- DeepSeek
- OpenRouter
- Claude
- Gemini
- Qwen
- Hunyuan

正式发布前需要在应用的开源许可/Notice 页面中补充 Lobe Icons 的 MIT License 信息。
